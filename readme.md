## Overview
**Project-VECNA** (Virtual Environment for Compromised Network Analysis) is an interactive, sentient terminal interface built with Python. What appears to be a standard "hacker-style" command line is actually a gateway to another dimension. 

This project explores **dynamic UI adaptation**, **event-driven animation**, and **interactive storytelling** through a terminal that "watches" the user and reacts to their input.

## Key Features
* **Sentient Eye Tracking:** A custom Canvas-based eye that follows the user's cursor in real-time.
* **Dimensional Breach:** When the signal counter reaches maximum (6), the system triggers a **physical screen shake** and a 180-degree layout inversion ("The Flip").
* **Glitch Engine:** Integrated text corruption logic that injects unpredictable symbols into system replies.
* **System Monitoring:** The terminal tracks idle time; if left unattended, it will initiate contact to ensure it is still being "watched."

## Technical Stack
* **Language:** Python
* **GUI Framework:** Tkinter
* **Data Storage:** JSON (for dialogue and glitch configuration)
* **OS Integration:** `platform` and `time` modules for environment-aware responses.
